개인정보처리방침

Privacy Policy
마지막 업데이트: 2026년 3월

1. 개요

소음 측정 기록기(이하 "앱")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보처리방침은 앱이 수집, 사용, 보호하는 정보에 대해 설명합니다.

2. 수집하는 정보

2.1 기기 정보

  • 마이크 접근을 통한 음성 데이터 수집
  • 기기의 센서 데이터 (오디오 레벨)

2.2 위치 정보

  • 측정 기록 시 선택적 주소 입력 (사용자가 직접 입력)
  • GPS 위치 데이터는 수집하지 않습니다

2.3 측정 데이터

  • 소음 레벨 (데시벨 값)
  • 측정 시간 및 기간
  • 등가소음도 (Leq) 계산 데이터
  • 사용자가 첨부한 이미지 및 메모

2.4 설정 정보

  • 기준 주소
  • 소음 기준값
  • 사용자 정의 태그

3. 정보 수집 목적

  • 소음 측정 및 기록 제공
  • 측정 통계 및 분석
  • 사용자 경험 개선
  • 데이터 백업 및 내보내기 기능 제공

4. 정보 보호 및 저장

4.1 로컬 저장소

  • 모든 측정 데이터는 기기의 로컬 저장소에만 저장됩니다
  • 클라우드 서버에 전송되지 않습니다
  • 사용자가 명시적으로 내보내기 또는 백업하지 않는 한 기기 외부로 전달되지 않습니다

4.2 보안

  • 기기의 운영 체제가 제공하는 보안 기능을 활용합니다
  • 저장된 파일은 기기의 접근 제어 메커니즘에 의해 보호됩니다

5. 데이터 접근 권한

앱이 요청하는 권한과 사용 목적:

권한 목적 필수 여부
마이크 소음 측정 필수
저장소 접근 이미지 첨부, 파일 내보내기 선택
위치 - 미사용

6. 파일 내보내기

사용자는 다음을 선택적으로 내보낼 수 있습니다:

  • 개별 측정 기록 (녹음, 이미지 포함)
  • 통계 분석 (CSV, PDF 형식)

내보낸 파일은 사용자의 선택한 저장소에만 저장되며, 앱은 이를 관리하지 않습니다.

7. 개인정보의 공유

  • 앱은 제3자와 개인정보를 공유하지 않습니다
  • 사용자가 명시적으로 파일을 내보내거나 공유하지 않는 한 데이터는 기기 내에만 존재합니다

8. 데이터 삭제

사용자는 언제든지 다음을 삭제할 수 있습니다:

  • 개별 측정 기록
  • 모든 측정 기록 및 관련 파일
  • 녹음 파일만 선택적 삭제
  • 이미지 파일만 선택적 삭제

삭제된 데이터는 복구할 수 없습니다.

9. 쿠키 및 추적

  • 앱은 쿠키를 사용하지 않습니다
  • 분석 또는 추적 도구를 사용하지 않습니다
  • 사용자 행동 데이터를 수집하지 않습니다

10. 미성년자 보호

앱은 13세 미만의 미성년자 개인정보를 의도적으로 수집하지 않습니다. 미성년자가 개인정보를 제공한 경우, 보호자에게 알려주시기 바랍니다.

11. 개인정보처리방침 변경

개인정보처리방침은 예고 없이 변경될 수 있습니다. 변경 시 앱 내에서 공지합니다.
