# Web Tools Landing Page

A simple, multilingual landing page showcasing web tools.

## Pages

- [English](index.html)
- [한국어](index_ko.html)

## Featured Tools

### PDF Clear
A web application to select specific areas of PDF files and fill them with a specified color. All processing happens in the browser with no server required.

[Visit PDF Clear](https://billyjodev.github.io/pdf-clear)

### MD Cleaner
A web-based tool to clean up and preview markdown documents from AI chat services (ChatGPT, Gemini, Claude, etc.) in real-time.

[Visit MD Cleaner](https://billyjodev.github.io/md-cleaner)

## 소개 도구

### PDF Clear
PDF 파일의 특정 영역을 선택하고 지정한 색상으로 채울 수 있는 웹 애플리케이션입니다. 모든 처리는 브라우저에서 이루어지며 별도의 서버가 필요 없습니다.

[PDF Clear 바로가기](https://billyjodev.github.io/pdf-clear)

### MD Cleaner
AI 채팅 서비스(ChatGPT, Gemini, Claude 등)에서 받은 마크다운 문서를 깔끔하게 정리하고 실시간으로 미리볼 수 있는 웹 기반 도구입니다.

[MD Cleaner 바로가기](https://billyjodev.github.io/md-cleaner)
